Custom Call To Action Gutenberg Block.

== Description ==
The Custom Call To Action Block for Gutenberg lets you add a call to action button in your posts or pages. It comes with a clean design and customization options like background colour, text colour, padding, font size etc. to suit your needs.


== Installation ==
1. Upload plugin to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.

== Screenshots ==
1. Open add new blocks from gutenberg editor and go to layout and add custom call to action block by just clicking on it.
2. Add button text, url etc. and create call to action button.
